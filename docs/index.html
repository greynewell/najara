Najara.py

Najara is a serverless Python/Flask based REST API for managing a collection of DND 5e items on AWS.

post /item/{id}
Create an item with the given id. (createItem)
Provide the necessary properties of the item and a valid, unused GUID id.

Path parameters

id (required)
Path Parameter — Desired id key of the item in GUID format

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json
  • application/xml

Request body

body Item (optional)
Body Parameter —

Return type

CreateSuccess

Example data

Content-Type: application/json
""

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json
  • application/xml

Responses

200

Success response- your item has been created! CreateSuccess

404

The given id is in use by another item. Please retry your request with a new id.

Example data

  "id" : "d66e22c7-95d2-49d2-95a2-77220a444411",
  "type" : "Ammunition"
}

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json
  • application/xml

Responses

200

404

Return type

    "name" : "Arrows",

    "id" : "d66e22c7-95d2-49d2-95a2-77220a444411"
  } ]
}
